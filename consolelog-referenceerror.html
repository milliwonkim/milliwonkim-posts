console.log에서 나는 ReferenceError는 뭐지?

생성일
Jun 26, 2023 10:24 PM

변수일 경우

console.log(message)// ReferenceError: message is not definedconst message = 'message'
만약 변수를 console.log한다면, console.log는 변수를 '참조한다' => 따라서 변수 선언 이전에 해당 변수를 참조하려했으므로
참조에러가 뜬다.

원시값인 경우

console.log(3)// primitive valueconsole.log("string")// primitive value
원시값이기 때문에 참조가 아니니, 할당이겠구나 싶겠지만
그건 아니다. 왜냐하면 변수에다가 값을 두는 건 아니고
단지 console.log에 값을 전달하기만 할 뿐이다.
